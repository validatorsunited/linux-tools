# linux-tools
Useful utilities for linux administration 

## Set swap
### Set swapfile to 16G
```
wget -q -O set_swap16.sh https://raw.githubusercontent.com/validatorsunited/linux-tools/main/set_swap16.sh && sudo chmod +x set_swap16.sh && sudo /bin/bash set_swap16.sh
```
